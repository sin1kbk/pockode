Why Pockode?

Mobile-first

Built for your phone. No tiny terminal fonts, no awkward keyboards — just tap, talk, and code.

Yours

Your code runs on your machine. Access it from anywhere via relay — no cloud sync, no third-party storage.

Instant

No setup, no sync. Run one command, scan the QR code, start coding from your phone in seconds.

Built-in

Git Worktree Management

Switch contexts without stashing. Work on multiple branches simultaneously with native Git worktree support — create, switch, and manage worktrees right from your phone.

  • Parallel branch development
  • Quick context switching
  • No more git stash chaos

Get Started

# Install
curl -fsSL https://pockode.com/install.sh | sh

# Run
pockode -auth-token YOUR_PASSWORD

Scan the QR code on your phone. That's it.
