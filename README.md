# Pockode

让手机编程不再痛苦。

## 项目愿景

手机编程一直是一个痛点：屏幕小、键盘不便、操作繁琐。Pockode 旨在彻底改变这一现状，提供一种全新的移动端开发体验。

## 核心理念

**AI 编辑为主，手动编辑为辅**

传统方案试图把桌面编辑器塞进手机屏幕，这是错误的方向。Pockode 采用对话式交互：

- 用自然语言描述你想要的改动
- AI 理解意图并执行代码修改
- 你只需审阅和确认

## 主要功能

### 对话式开发
通过 Chat 界面与 AI 交互，完成大部分开发工作：
- 编写和修改代码
- 重构和优化
- Debug 和问题排查
- 代码解释和学习

### Git 操作
不再需要记忆命令，用自然语言完成版本控制：
- "提交这些改动"
- "创建新分支 feature/login"
- "合并到主分支"
- "查看最近的提交记录"

### 手动编辑
当你确实需要手动编辑时：
- 语法高亮
- 快捷操作面板
